# Performance

Keep the POC fast and lightweight. Avoid large client libraries, chart libraries, animation libraries, state libraries, and PDF libraries unless a later module audits and approves them.

Landing pages should feel instant on normal 4G, quiz transitions should feel immediate, result generation should complete quickly, and admin dashboard views should remain usable with hundreds of demo leads.

Use route-level code splitting, server rendering where practical, optimized images, simple CSS/SVG before heavy libraries, and lazy-load admin/demo-only heavy surfaces.

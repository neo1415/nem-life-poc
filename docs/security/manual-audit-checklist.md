# Manual Audit Checklist

Customer journey:

- [ ] No contact capture before result.
- [ ] No BVN/NIN/payment/upload/policy number/exact address.
- [ ] No fake verification or fear language.

Lead capture:

- [ ] Consent visible and unchecked by default.
- [ ] Contact purpose clear.
- [ ] Contact data validated and minimized.

Report/dashboard:

- [ ] Disclaimers visible.
- [ ] Print/save output excludes internal metadata.
- [ ] Email preview does not claim delivery.
- [ ] Future verified placeholders labeled.

Admin/config:

- [ ] Demo warning visible.
- [ ] Contact masked.
- [ ] No production CRM/security claim.
- [ ] Export safe and demo-labeled.
- [ ] Unsafe config blocked.

Demo/build:

- [ ] Fictional data labels visible.
- [ ] Reset preserves unrelated storage.
- [ ] Typecheck, lint, tests, build, audit, E2E pass.
